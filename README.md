Error Makes Clever Academy – Web Page

This project is a simple educational web page created using HTML in Visual Studio Code. The website introduces the concept of full stack development and includes an embedded video, technology overview, and a course registration form.

Project Description

The purpose of this project is to practice and understand the basic structure of web pages using HTML. In this website, I implemented different HTML elements such as headings, paragraphs, tables, forms, links, and embedded media.

A YouTube video about full stack development is included using the iframe tag. The video is added only for learning purposes to understand how embedding external content works in a webpage.

 Features

Header section introducing Error Makes Clever Academy

Embedded YouTube video using iframe

Information about full stack development

Table showing technologies used in frontend, backend, and database

Course registration form with user input fields

Navigation links for Home, Instagram, and YouTube
